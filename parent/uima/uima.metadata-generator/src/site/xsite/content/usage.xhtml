Usage: How to use the UIMA metadata generator

Author: Tommaso Teofili

Date: July 27, 2010

1. Install UIMA bundles

2. Install Clerezza - UIMA utils bundles

3. Install Metadata Generator bundle

Now you can use the UIMABaseMetadataGenerator, for example you could bind such a generator to Discobit Editor to enable automatic extraction of the graph of tags and entities for a document.
