Usage: How to use the User Manager

Author: Daniel Spicar, Tsuyoshi Ito - clerezza.org

Date: March 25, 2009

1. Add a new user

In order to add a new user to the CMS click the Add User link in the user list (http://localhost:8282/admin/user-manager/). Fill in the form with the Username, E-Mail address and the ID and a password. Then choose a role and click the add user button.

To add some special information to the user profile, single and multiple informations can be added to the form by clicking the corresponding buttons. Keep attention that the property identifier is a valid URI. Otherwise the graph will be compromised.

2. Manage permissions

The user permissions can be edited by clicking the manage permissions button. On the following page the current user permissions are listed. For example the admin has the entry (java.security.AllPermission "" ""). These permissions are based on Java Authentication and Authorization Service (JAAS). More information on how to use them can be found at: JAAS Permissions.

To add a new permission enter it into the text field and press the add new permission button.

To remove existing permissions find the permission in the list and click the delete button.

3. Delete a user

Deleting a user is very simple. Just click the delete button in the user-list and the it will be deleted.

4. Edit a user

The first step of editing a user is done by clicking on the edit button in the user-list.On the following page the current user data is showed, and it is possible to add some new data to the appropriate fields. By clicking the save changes button, all the entered data will be stored to the user.

5. Manage User Formfields

On the management site for user formfields (http://localhost:8282/admin/user-manager/manage-user-form-fields) you can delete defined properties or define others according to the add-user page.

To remove existing formfields find the formfield in the list and click the delete button.
