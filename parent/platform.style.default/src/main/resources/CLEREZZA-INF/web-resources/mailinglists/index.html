Mailing lists

The following mailing lists have been established for Apache Clerezza. For each list, there is a subscribe, unsubscribe, and archive link.


Name Subscribe Unsubscribe Archive
Clerezza Developers - For people using and developing Clerezza:
clerezza-dev at incubator.apache.org		 Subscribe Unsubscribe Mail Archive
Clerezza Commits - For people interested in code changes:
clerezza-commits at incubator.apache.org		 Subscribe Unsubscribe Mail Archive


Note: please don't send patches or attachments to any of the mailing lists. Patches are best handled via the Issue Tracking system. Otherwise, please upload the file to a public server and include the URL in the mail.
