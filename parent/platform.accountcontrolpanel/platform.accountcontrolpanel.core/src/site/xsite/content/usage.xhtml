Usage: How to use the Account Control Panel

Author: Tsuyoshi Ito - clerezza.org

Date: January 27, 2009

1. List installed bundles

Each user can upload and install bundles into the platform. Users and permissions can be created via http://localhost:8282/admin/user-manager and http://localhost:8282/admin/role-manager.
Enter the following URL: http://localhost:8282/user/admin/control-panel to list all bundles from the user "admin" (when asked to authenticate use username "admin" and password "admin). If you want to list installed bundles from another user, replace "admin" in the URL with the username of the other user.

2. Install a bundle

You can upload a bundle by clicking on the link Install Bundle on http://localhost:8282/user/admin/control-panel. Enter the URL of your bundle (e.g. Location: http://localhost/mybundle-1.0.jar) and click on the button "install". The uploaded and installed bundles have the same permission as you have.

3. Start, Stop and Uninstall a bundle

A user can start, stop and uninstall a bundle on http://localhost:8282/user/admin/control-panel, whereas "admin" can be replaced by another username. Click on the appropriate button for start , stop or uninstall a bundle.

That's all. Enjoy.
