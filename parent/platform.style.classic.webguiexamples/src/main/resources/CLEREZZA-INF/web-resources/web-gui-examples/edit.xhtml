Description

In the tx-content a <div> with classtx-edit is created. Within div.tx-edit a div.tx-edit-content is added.

li.tx-line draws a line under the <li> element.

The current example has no ids or names defined, if necessary they have to be defined contextually.
div.tx-edit is the main container for the edit view. If no class is defined on the div.tx-edit element the edit view has 100% width.
The classes .tx-show-panel, .tx-hide-panel and .tx-halfwidth-right define the position of the edit view.

.tx-show-panel: position with a panel on the left side.
.tx-hide-panel: position with a panel on the left side, hidden.
.tx-halfwidth-right: position on the right, 50% width.
