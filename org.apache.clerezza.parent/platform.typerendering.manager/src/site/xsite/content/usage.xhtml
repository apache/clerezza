Usage: How to use the the Renderlet Manager

Author: Tsuyoshi Ito - clerezza.org

Date: May 5, 2009

1. List Renderlets

The renderlet manager allows renderlets for rendering resources of a certain type to be defined as well as setting and changing the priority of the renderlets. An overview of the installed renderlets can be found under http://[your.server]/admin/renderlet-manager/overview.
The renderlets are prioritized (The topmost is the first renderlet of the list). Use the links to "up" or "down" to change the order.

2. Upload a Renderlet

You can install a renderlet via http://[your.server]/admin/renderlet-manager/install-renderlet-form.

That's all. Enjoy.
