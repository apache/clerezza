Seedsnipe - Simple templating languag

Seedsnipe is meant to be easy and higly intuitive, so a couple of examples should do :).

A nested loop, at least one of the variables is an array of two (or more) dimenstions.

${loop} ${loop} a variable: ${var1} another variable: ${var2} ${/loop} ${/loop}

Display a foaf:PersonalProfileDocument (as HTML snippet):

${loop} Name: ${foaf:primaryTopic/foaf:name}<br /> Phone: ${foaf:primaryTopic/foaf:phone}<br /> Friends:<br/> ${loop} ${loop} ${if foaf:primaryTopic/foaf:knows/rdf:type="<http://xmlns.com/foaf/0.1/Person>"} Person: ${else} Agent: ${/if} ${foaf:primaryTopic/foaf:knows/foaf:name} <br/> ${/loop} ${/loop} ${/loop}

Sorting loops in acsending and descending order:

${loop sort desc outer} ${loop sort asc inner} a variable: ${outer} another variable: ${inner} ${/loop} ${/loop}
