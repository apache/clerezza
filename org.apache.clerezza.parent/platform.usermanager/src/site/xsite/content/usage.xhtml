Usage: How to use the user manager service

Author: Tsuyoshi Ito - clerezza.org

Date: May 27, 2009

Service

This service provides functions for manipulating user information. The following list specifies the available functions:

  • store roles
  • check if a role already exists
  • get role by title
  • get all roles (NonLiterals)
  • delete roles
  • assign permissions to a role
  • get permission of a role
  • delete permission of a role
  • delete all permissions of a role
  • store users
  • update users
  • update properties
  • check if name already exists
  • check if email already exists
  • get name by email
  • get users
  • delete users
  • assign permissions to user
  • get permissions of user
  • delete permissions of user
  • delete all permissions of user
