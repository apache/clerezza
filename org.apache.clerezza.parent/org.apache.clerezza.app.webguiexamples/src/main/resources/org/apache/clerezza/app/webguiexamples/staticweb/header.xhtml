Description

The header is a <div> with id tx-header and is the first child of the <body>.

The div#tx-system contains the globalsearch, menu, logo and login.
The global search is a <div> with id tx-globalsearch and contains HTML.
The logo could be placed on the left top corner and is a <div> containing a <a> e.g. <div id="tx-logo"><a href="" /></div>
The menu is a <div> with id tx-menu, which contains an ordered list.


The div#tx-module contains the title and the page actions.
The title of the module is a <h1>. The page action buttons area is a <div> with id tx-page-actions, which contains an ordered list.


The tabs and contextual buttons are in a <div> with id tx-module-tabs which contains:
An ordered list <ol> for the module tabs.
A <div> with id tx-contextual-buttons, which contains an ordered list of contextual action buttons


The div#tx-module-options contains the toggle icons and paginator/select elements.
An ordered list <ol> for the module options, which contains the toggles and status icons.
